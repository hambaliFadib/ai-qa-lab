# 06 Testing

Execution buckets for human and agent-driven test work.

- `smoke/` stores stable sanity checks.
- `exploratory/` stores guided exploration assets.
- `adhoc/` stores temporary one-off probes.
- `uat-draft/` stores draft UAT scenarios.

Keep tests lightweight and module-aware; move obsolete drafts to `99-archive/` instead of deleting them blindly.
