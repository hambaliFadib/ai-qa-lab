# QA Standards

This folder stores reviewed QA governance summaries.

Reviewed governance may guide FALID behavior for evidence discipline, testcase quality, defect triage, release readiness, stakeholder communication, and intern or team task delegation.

These summaries must not override project-specific MoM, BPMN, Figma, UI actual behavior, controlled testcase source, or stakeholder confirmation. When sources conflict, FALID must route the finding to reconciliation and confirmation before treating it as truth.
