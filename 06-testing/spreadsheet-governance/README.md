# Spreadsheet Governance

TEST CASE - PGN BILLING is a controlled QA execution source.

FALID may read the spreadsheet to understand test scenarios, test cases, status, actual result, issue URL, owner, and execution notes. FALID may propose updates when evidence shows a testcase is outdated, incomplete, or missing execution context.

FALID must not directly update the spreadsheet without QA approval. Spreadsheet writes must use an explicit approval gate. Every update must show old value, new value, source evidence, confidence, and reason before any write command is allowed.
