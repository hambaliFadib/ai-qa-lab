# OpenCode Brain Notes

This `.opencode` tree is the active brain package for the PGN Billing QA workspace.

- `agents/` contains the unified Engineer operating role.
- `prompts/` contains reusable execution prompts.
- `skills/` contains disk-based rules that should still be used even if skill registration is unavailable.
- `memory/` contains active durable memory.
- `knowledge/` contains reusable cross-run guidance.
- `api-discovery/` contains discovered endpoint notes.
