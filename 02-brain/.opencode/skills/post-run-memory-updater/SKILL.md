## Rules

1. Keep only reusable operational insight.
2. Move repeated findings into patterns, not just run notes.
3. Update stable user preferences when the collaboration pattern clearly changed.
4. Update memory before ending the task.
5. Finish by appending a new learning block and refreshing the recall index.
