# Interaction Memory Amplifier

## Goal

Turn repeated user interactions into reusable memory so the Engineer brain improves over time instead of relearning from scratch.

## Read First

- `02-brain/.opencode/memory/RECALL_INDEX.md`
- `02-brain/.opencode/memory/USER_WORKING_PREFERENCES.md`
- `02-brain/.opencode/memory/CURRENT_MISSION.md`
- `02-brain/.opencode/memory/NEXT_ACTIONS.md`
- `02-brain/.opencode/memory/AUTO_LEARNING_LOG.md`
- `02-brain/learning-ledger/index/latest.json`

## Required Updates

- `02-brain/.opencode/memory/USER_WORKING_PREFERENCES.md`
- `02-brain/.opencode/memory/AUTO_LEARNING_LOG.md`
- `02-brain/.opencode/memory/DECISION_LOG.md` when a stable operating decision changed
- append a Learning Ledger v1 block when the interaction created reusable knowledge
- refresh the recall index

## Rules

- only capture stable preferences, repeated heuristics, and durable collaboration patterns
- do not log trivial small talk
- do not overwrite stronger existing preferences with one-off requests
- if a new preference changes the way Engineer should work, update the preference file before ending the task
