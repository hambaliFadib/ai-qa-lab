# App Testing Standards Raw

Raw application-specific testing standards for PGN Billing.

Use this folder for rules that belong to this product and should stay separate from generic QA standards, for example:

- currency and amount precision
- date or period formatting
- status wording
- list, action-column, and search-column behavior
- UI evidence rules tied to Ant Design or this application shell

Distill confirmed or repeated rules into `02-brain/distilled-output/global/app-specific-testing-standards.md` after they become useful for daily testing.
