# ISQA Raw Knowledge

This folder stores raw ISQA conference material manually added by the user.

Raw material is not durable truth by default. ISQA material must be extracted, summarized, reviewed, and approved before it is used as a FALID governance rule.

ISQA is a QA governance reference. It can improve evidence discipline, testcase quality, defect triage, release readiness thinking, stakeholder communication, and team delegation. It is not project-specific business truth and must not define PGN Billing behavior by itself.
